# 🛒 Ecommerce Fullstack Application

## 📌 Project Description
This is a full-stack ecommerce web application built using Spring Boot and React. It allows users to browse products, manage cart, and perform CRUD operations.

---

## ⚙️ Tech Stack

### Backend
- Spring Boot
- Spring Data JPA (Hibernate)
- MySQL
- REST APIs

### Frontend
- React
- Axios
- Bootstrap / Tailwind

---

## 🔑 Features
- User authentication (JWT)
- Product management
- Cart functionality
- CRUD operations
- API integration

---

## 🗄️ Database
- MySQL database
- Entity relationships (One-to-Many, Many-to-Many)

---

## 🚀 How to Run

### Backend
1. Open project in IDE
2. Run Spring Boot application
3. Configure `application.properties`

### Frontend
1. Go to frontend folder
2. Run:
   ```bash
   npm install
   npm run dev
